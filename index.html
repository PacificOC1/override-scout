Points calculation

Alliance Score = (T1_G1 + T1_G2 + T2_G1 + T2_G2) × 5 Consistency = (1 - T1_FailRate + 1 - T2_FailRate) / 2 ExpectedPts = Alliance Score × Consistency AutoBonus = +12 pts to higher-scoring alliance (SC7) +6 to each alliance if tie

AWP probability

AWP% = (R1_awp/100) × (R2_awp/100) × ConsFactor × 100 Both robots must succeed — joint probability. AWP earns WP (ranking pts), NOT match score.

Worked example

Red: R1 → 4 pins, 80% AWP, cons 4/5 | R2 → 3 pins, 60% AWP, cons 3/5 RawPts = 7×5 = 35 | ConsFactor = 0.70 | ExpPts = 24.5 AWP% = 0.80×0.60×0.70×100 = 33.6% Blue: R1 → 5 pins, 40% AWP, cons 5/5 | R2 → 5 pins, 70% AWP, cons 4/5 RawPts = 10×5 = 50 | ConsFactor = 0.90 | ExpPts = 45.0 AWP% = 0.40×0.70×0.90×100 = 25.2% Blue wins points (45 > 24.5) → Blue gets +12 auto bonus Final: Red 24.5 pts, Blue 57.0 pts Red still more likely to achieve AWP (33.6% vs 25.2%)
