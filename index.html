Points calculation

RawScore = (R1_G1 + R1_G2 + R2_G1 + R2_G2) × 5 AutoBonus = +12 pts to higher-scoring alliance (SC7) +6 to each alliance if tied on raw score FinalScore = RawScore + AutoBonus

Win probability (16-outcome model)

Each robot independently succeeds or fails each match. All 16 combinations (2⁴) are enumerated. For each outcome: P(outcome) = ∏ P(robot_i succeeds or fails) P(succeed) = 1 − FailRate P(fail) = FailRate Scores are computed per outcome, then the bar segments are sized proportionally to P(outcome). Red win % = Σ P(outcomes where redScore > blueScore) Blue win % = Σ P(outcomes where blueScore > redScore) Tie % = Σ P(outcomes where scores are equal)

AWP probability

AWP% = R1_awp × R2_awp (per outcome, if both succeed) Both robots must succeed AND individually earn AWP. AWP earns WP (ranking pts), NOT autonomous match score.

Worked example

Red: R1 → 4 pins total, 0% fail | R2 → 3 pins total, 0% fail RawScore = 7 × 5 = 35 Blue: R1 → 5 pins total, 0% fail | R2 → 5 pins total, 0% fail RawScore = 10 × 5 = 50 Blue wins raw (50 > 35) → Blue gets +12 auto bonus Final: Red 35 pts, Blue 62 pts With 0% fail rates → P(both succeed) = 100% → Blue wins probability = 100% in this outcome
